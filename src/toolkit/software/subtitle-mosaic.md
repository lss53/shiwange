1. 打开视频，拖动进度条到需要开始截图的位置。
2. 在视频画面上**右键点击** → 选择 **「视频」→「图像截取」→「连续截图」**，或直接使用快捷键 `Ctrl + G`。
3. 在弹出的 **连续截图画面设置** 窗口中配置关键参数（其他保持默认）：
   - **保存路径**：新建一个文件夹，例如 `D:\SubtitleScreenshots`
   - **文件前缀**：方便统一命名，如 `Sub_`
   - **采集数量**：视台词长度而定，建议 `100`（自由修改）
   - **截取间隔**：建议选择 **「按帧间隔」**，数值设为 `1`（自由修改）
4. 点击 **「开始」** 按钮，然后回到视频播放界面点击 **播放按钮（三角形图标）**。
5. 视频播放过程中，截图窗口会自动抓取。当截够所需数量后，点击 **连续截图画面设置** 窗口中的 **「停止」** 按钮和点击 **「打开存档文件夹」** 即可查看所有截好的图片。
6. 将需要的截图用[截图拼接工具](https://join-screenshots.zhanghai.me)（**推荐**）或[电影字幕拼接工具](http://screenshots.hpday.cn)拼接成一张完整的拼图。
