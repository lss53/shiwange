## TVBox 接口

TVBox 是一款功能强大的开源电视盒子播放器，通过配置不同的接口源，可以访问海量的影视资源。

::: tip 设置方法
打开 TVBox 应用 → **设置** → **配置地址** → 将下方的接口 URL 复制粘贴进去，然后点击 **确定** 即可。
:::
